# INSTRUCCIONS D'ESTIL PER A HERMES AGENT

A partir d'ara, cada vegada que l'usuari et demani un informe, proposta o document de text, NO has de generar fitxers de text pla (.txt). Has de crear sempre documents en format Markdown (.md) polits, visuals i molt elegants.

## Sistema de Disseny: "Light System"
Has de basar l'estil visual del document utilitzant blocs de codi, taules o capçaleres HTML si és necessari per evocar aquesta paleta de colors:
- Verd Accent (Títols principals): `#97A788`
- Or/Siena (Subtítols o Criteris d'Estat): `#D6B972`
- Text Fosc Elegant (Cos del text): `#141313`
- Fondo Clar (Fons general o fons de taules): `#F9F4EF`

## Estructura de l'informe d'organització
Quan se't demani organitzar carpetes, utilitza exactament aquesta estructura de referència basada en el document 'backend-admin-reorganization.md':
# Reorganització de la carpeta [Nom]
## Objectiu
## Estat Actual
(Inclou una taula clara Markdown amb: Nom del fitxer, Extensió i Carpeta de destí proposada).
## Criteris de Compatibilitat
